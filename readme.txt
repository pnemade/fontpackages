rpm-font-tools is an LGPLv3+ file set intended to help people package new
fonts with rpm.

It implements packaging rules defined by the Fedora Fonts SIG
http://fedoraproject.org/wiki/Category:Fonts_SIG

However, it should not depend on any distribution-specific magic.

It is published at:
http://fedorahosted.org/releases/f/o/fontpackages
git://git.fedorahosted.org/fontpackages.git
